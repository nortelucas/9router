Recursos principais

Tudo o que você precisa para gerenciar sua infraestrutura de IA em um só lugar.

🔀

Roteamento inteligente

Combos com estratégia de fallback: se um provedor atingir o limite ou falhar, a requisição segue automaticamente para o próximo — sem interromper seu fluxo de trabalho.

🔐

Login OAuth em 1 clique

Conecte contas gratuitas sem copiar chaves de API: Qwen, iFlow, Gemini CLI, Copilot, Kiro, Antigravity, Grok, Kimi, Cline, KiloCode e muito mais.

💰

Economia de tokens (RTK)

Compressão inteligente de contexto que reduz o consumo de tokens em até 40%, com modos configuráveis por perfil de uso.

📊

Dashboard completo

Monitore uso, custos, quotas e histórico de requisições de todos os provedores em tempo real, com detalhes por modelo e por conexão.

🔄

Tradução de protocolos

Recebe requisições nos formatos OpenAI, Claude ou Gemini e traduz automaticamente para qualquer provedor de destino — compatível com qualquer ferramenta.

🛠️

Integração com CLIs

Configuração assistida para Claude Code, Codex CLI, OpenCode, Qwen Code, Droid e outros — direto do dashboard, com um clique.

Como começar

Exemplo de configuração do OpenCode com o MaxRouter:

// ~/.config/opencode/opencode.json
{
  "provider": {
    "maxrouter": {
      "npm": "@ai-sdk/openai-compatible",
      "options": {
        "baseURL": "https://maxrouter-prod.up.railway.app/v1",
        "apiKey": "sua-chave-aqui"
      }
    }
  }
}
